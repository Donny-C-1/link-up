## Installation

To get started with the Chat App, follow these steps:

1. **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/chat-app.git
    ```
2. **Navigate to the project directory:**
    ```bash
    cd chat-app
    ```
3. **Install dependencies:**
    ```bash
    npm install
    ```
4. **Start the development server:**
    ```bash
    npm start
    ```

## Usage

Once the server is running, open your browser and navigate to `http://donny-c-1.github.io/link-up` to start using the Chat App. You can create an account, log in, and start chatting with other users.
